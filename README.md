# llm-from-scratch-study-notes

## 仓库说明
本仓库为 Sebastian Raschka 著作 **《Build a Large Language Model From Scratch》**（中文译名《从零构建大模型》）的个人学习复现仓库。
核心模型逻辑基于原书官方代码与社区中文汉化版本整理，仅用于个人学习沉淀与知识分享，不涉及任何商业用途。

- 原书官方代码仓库：[rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch)
- 参考中文汉化版本：LLMs-from-scratch-CN

## 个人学习产出
本仓库为个人随书体系化学习的沉淀产物，所有增量内容均为独立手写实现，在原书代码基础上完成了注释完善与内容拓展：

1.  **全部代码注释中文化**：覆盖所有主章节、附录、习题解答与辅助工具代码，填补原版中非主章节无中文注释的空白，完成全仓库代码注释的完整中文适配。
2.  **主章节逐行级原理注释增强**：核心章节在基础注释之上大幅扩充底层原理讲解，标注张量形状变化、注意力计算逻辑、模块间数据流转、内存占用变化等关键细节；核心代码段近乎逐行配套解释，注释密度与深度远超原版，强化原理与代码的对应理解。
3.  **前置知识体系补全**：针对原书第1章只有理论、无实践代码的空白，补充了从静态词嵌入到Transformer编码器的完整前置实践，搭建完整入门链路：
    - Word2Vec Skip-Gram 三种梯度实现（纯Python手写版 / NumPy向量化版 / PyTorch自动微分版）
    - 从零手写Transformer编码器文本分类模型，直观验证动态词嵌入与自注意力的核心特性
4.  **推理优化扩展实验（持续更新）**：在原生GPT模型基础上拓展底层推理优化，每个优化点均完成「原理实现 → 正确性校验 → 性能基准测试」的闭环：
    - 已完成：基础版 KV Cache 推理缓存，1024上下文长度下后续token推理加速可达20倍以上
    - 规划中：FP16混合精度推理、量化推理、C语言无框架GPT-2复现、CUDA算子级推理优化

> 本仓库对应个人学习路径中的「教材体系化筑基」环节，从底层词嵌入基础到核心GPT模型，再到推理优化拓展，形成完整的学习路径，后续将向C/CUDA底层优化方向持续进阶。

## 原书简介
《Build a Large Language Model From Scratch》是大模型底层学习领域的经典实践教材。全书避开高层框架的黑盒封装，从最基础的文本处理、注意力原理出发，带领读者逐行手写实现完整的GPT类大语言模型，覆盖模型架构、预训练、下游任务微调全流程，是深入理解Transformer内部运行机制的优质学习材料。
