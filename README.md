# llm-from-scratch-study-notes

## 仓库说明
本仓库为 Sebastian Raschka 著作 **《Build a Large Language Model From Scratch》**（中文译名《从零构建大模型》）的个人学习复现仓库。
核心模型逻辑基于原书官方代码与社区中文汉化版本整理，仅用于个人学习沉淀与知识分享，不涉及任何商业用途。

- 原书官方代码仓库：[rasbt/LLMs-from-scratch](https://github.com/rasbt/LLMs-from-scratch)
- 参考中文汉化版本：LLMs-from-scratch-CN

## 个人学习产出  
本仓库为个人随书体系化学习的沉淀产物，在原书官方代码与汉化版本的基础上，完成了以下增量工作：  
1. **全量代码注释中文化**：覆盖所有主章节、附录、习题解答与辅助工具代码，填补原版中非主章节无中文注释的空白，实现全仓库代码注释的完整中文适配。  
2. **主章节逐行级原理注释增强**：核心章节在原有中文注释基础上大幅扩充底层原理讲解，标注张量形状变化、注意力计算逻辑、模块间数据流转、内存占用变化等关键细节；核心代码段近乎逐行配套解释，注释密度与深度远超原版，强化原理与代码的对应理解。 
3. **推理优化扩展实验（持续更新）**：后续将逐步补充 KV Cache 推理缓存、FP16 混合精度推理、性能基准测试等内容，作为底层推理优化的 Python 基线验证。

> 本仓库对应个人学习路径中的「教材体系化筑基」环节，与后续纯C语言无框架GPT-2复现、CUDA算子级推理优化项目形成完整进阶链路。

## 原书简介
《Build a Large Language Model From Scratch》是大模型底层学习领域的经典实践教材。全书避开高层框架的黑盒封装，从最基础的文本处理、注意力原理出发，带领读者逐行手写实现完整的GPT类大语言模型，覆盖模型架构、预训练、下游任务微调全流程，是深入理解Transformer内部运行机制的优质学习材料。
