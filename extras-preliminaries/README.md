# 前置知识补全（Preliminaries）
## 目录说明
属于我个人补充的拓展代码，弥补原书第一章只有理论无实践的短板，搭建「静态词嵌入→Transformer动态上下文向量」前置学习链路，用来铺垫后续GPT模型学习。
全部代码为教学极简手写实现，侧重底层数学推导与张量流转，无工业封装。

## 1. Word2Vec Skip-Gram 三套实现
分三层递进写法，从纯手写梯度到框架自动微分，循序渐进搞懂词向量训练逻辑。
| 文件 | 依赖 | 核心作用 |
| ---- | ---- | -------- |
| Word2Vec_python.py | 标准库 | 手动实现前向传播、损失、反向梯度，完全对应数学公式 |
| Word2Vec_numpy.py | NumPy | 向量化重构，对比纯循环，直观体现矩阵运算提速优势 |
| Word2Vec_pytorch.py | PyTorch | 框架标准写法，对比手动梯度，理解自动微分机制 |

### 核心结论
训练后同类词语余弦相似度更高，验证分布式语义假设：词的语义由上下文决定。

## 2. 手写Transformer编码器分类模型
独立复现全套Transformer编码器组件，用二分类任务直观对比Word2Vec静态向量与动态上下文向量的差异。
### 实现要点
- 从零手写位置编码、多头注意力、FFN、残差&层归一化，不调用官方现成Transformer接口
- 内置注意力权重可视化，直观查看模型学到的词语关联
- 完整训练+评估流程，一键复现多义词向量变化演示

### 核心结论
自注意力带来上下文动态词表征，解决Word2Vec静态向量无法区分多义词的短板，是GPT模型的底层基础。
