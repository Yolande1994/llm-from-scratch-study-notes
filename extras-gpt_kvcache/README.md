# GPT-2 KV Cache 推理优化实现
## 目录说明
个人原创的推理优化拓展代码，基于仓库原生GPT-2 124M改造，属于原书内容之外的进阶性能实验，用来吃透大模型增量推理的核心优化逻辑。

## 实现要点
- 逐层改造注意力、Transformer块与生成函数，打通KV缓存传递链路：预填充阶段一次性算完全文KV，后续每步仅输入单token复用历史缓存，省去重复全量注意力计算。
- 内置正确性校验，生成结果和原版无缓存完全一致，无精度损失。
- 自带标准化基准测试，自动统计多上下文长度下的首token延迟、单token平均耗时、加速比。
- 兼容CPU/GPU，带基础滑动窗口截断逻辑，全代码配中文注释标注关键细节。

## 实测效果（CPU环境，生成32token）
| 上下文长度 | 后续token加速比 |
| :---: | :---: |
| 256 | 7.16x |
| 512 | 12.56x |
| 1024 | 20.27x |

- 首token为预填充阶段，两者计算量相当，缓存版无明显加速
- 上下文越长加速收益越显著，长文本场景下后续token推理提速超20倍
- 显存指标需GPU环境查看，本次CPU测试未统计
